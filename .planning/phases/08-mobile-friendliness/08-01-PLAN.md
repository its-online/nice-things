### Task 4: Add Mobile Filter Toggle JavaScript

**Requirements:** MOBILE-04

**Read first:**
- `src/pages/index.astro` (lines 110-130 for script variables)
- `src/pages/index.astro` (lines 295-310 for existing event listeners)

**Action:**
Add the following JavaScript after the existing script variables:

```javascript
const filterToggle = document.getElementById('filter-toggle');
const filterDropdown = document.getElementById('filter-dropdown');

function isMobile() {
  return window.innerWidth <= 640;
}

function toggleFilterDropdown() {
  if (isMobile()) {
    filterDropdown.classList.toggle('mobile-open');
  }
}

if (filterToggle) {
  filterToggle.addEventListener('click', toggleFilterDropdown);
  
  filterDropdown.addEventListener('click', (e) => {
    if (e.target === filterDropdown) {
      filterDropdown.classList.remove('mobile-open');
    }
  });
}
```

Update the existing `filter-select:focus-within` logic to handle mobile:
Add this after the new mobile JavaScript:

```javascript
window.addEventListener('resize', () => {
  if (!isMobile()) {
    filterDropdown.classList.remove('mobile-open');
  }
});
```

**Acceptance criteria:**
- JavaScript contains `filterToggle` variable getting element by `filter-toggle`
- JavaScript contains `isMobile()` function checking `window.innerWidth <= 640`
- Click handler toggles `mobile-open` class on mobile
- Window resize listener removes `mobile-open` class when switching to desktop
